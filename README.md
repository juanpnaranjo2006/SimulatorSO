# 🖥️ SimulatorSO

Simulador de Sistema Operativo compilado en C++.

---

## 🚀 Ejecutable listo para usar

El ejecutable final del proyecto se llama:

```
SimulatorSO.exe
```

No requiere el IDE para ejecutarse. Solo debes abrirlo desde la terminal o hacer doble clic (aunque se recomienda usar terminal para pasar argumentos).

---

## ▶️ Formas de ejecución

El programa puede ejecutarse de **dos maneras**:

### 1️⃣ Con argumento por terminal

Puedes pasar la ruta del programa directamente al ejecutar el `.exe`:

```bash
SimulatorSO.exe testPrograms/program.txt
```

---

### 2️⃣ Por entrada de texto una vez iniciado

También puedes ejecutar:

```bash
SimulatorSO.exe
```

Y luego, cuando el programa lo solicite, escribir manualmente la ruta del archivo:

```
testPrograms/program.txt
```

---

## 📂 Ejecutar programas de prueba

Para simular un programa:

1. Ve a la carpeta:

```
testPrograms
```

2. Escoge uno de los archivos `.txt` disponibles.

3. Escribe la ruta en el siguiente formato:

```bash
testPrograms/nombreDelPrograma.txt
```

📌 Ejemplo:

```bash
SimulatorSO.exe testPrograms/ejemplo1.txt
```

---

## ⚠️ Importante

* Asegúrate de ejecutar el `.exe` desde la carpeta raíz del proyecto.
* La ruta debe escribirse exactamente como:

```
testPrograms/program.txt
```

De lo contrario, el archivo no podrá ser encontrado.

---
